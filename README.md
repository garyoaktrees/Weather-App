# Weather App

A desktop weather app built with Python and PySide6.

## Features

- City search
- Weather data display
- Autocomplete support
- Custom UI components
- Settings stored with JSON
- Weather icons and background images

## Project Structure

- `main.py` - app entry point
- `app/` - main application code
- `data/` - city data
- `icons/`, `images/`, `alerts/` - visual assets

## Tech Used

- Python
- PySide6
- JSON

## Notes

This is an early working version of the project. I plan to keep improving it over time.
